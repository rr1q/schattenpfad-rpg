# ⚔️ Schattenpfad RPG

Ein rundenbasiertes RPG mit Raritätssystem, Turm, Quests und Edelsteinen.
Entwickelt in Python mit Pygame.

## ✨ Features
- 🗡️ 3 Klassen: Krieger, Magier, Schurke
- 💎 6 Raritätsstufen (Common bis Mythic)
- 🏰 Turm mit dynamisch skalierten Ebenen
- 📜 Quest-System mit 10 Quest-Typen
- 💾 Speichern & Laden (3 Slots + Autosave)
- ⚡ Wiederbeleben-System

## 🚀 Installation
```bash
git clone https://github.com/DEIN-NAME/schattenpfad-rpg.git
cd schattenpfad-rpg
pip install -r requirements.txt
python main.py
```
